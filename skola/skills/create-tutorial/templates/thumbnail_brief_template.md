## Production Specs

### Canvas Size
- **1280x720px** (16:9 aspect ratio)
- YouTube recommended resolution

### File Format
- **JPG** (smaller file size, good for photos/gradients)
- **PNG** (larger file size, better for sharp text/graphics)

### File Size
- Under 2MB (YouTube limit)
- Optimize for web without quality loss

### Safe Zones
- Keep critical text/elements away from edges
- Leave 10-20px margin from all sides
- Account for player controls overlap in lower portion

---

## Accessibility

### Colorblind-Friendly
- Test with colorblind simulator
- Don't rely solely on color to convey meaning
- Use shapes, contrast, and text

### Text Readability
- Minimum font size: 48pt on 1280x720 canvas
- High contrast: 4.5:1 or better
- Avoid decorative fonts that are hard to read
