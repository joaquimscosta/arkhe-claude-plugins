## Pacing Guide

**Overall Pace:**
- Hook & Agenda: Fast
- Context: Moderate
- Setup: Fast (defer details to description)
- Main sections: Moderate to fast (but clear)
- Recap: Moderate
- Outro: Fast

**Energy Levels:**
- High: Hook, demo moments, recap, CTA
- Moderate: Explanations, implementation
- Calm: Troubleshooting

## Post-Production Notes

### Editing Points
- Speed up repetitive typing (1.2-1.5x)
- Add zoom/highlight on complex code
- Insert B-roll during explanations (if available)
- Background music: subtle, low volume during implementation

### On-Screen Elements
- Consistent style for callouts and text
- Code highlights with arrows or boxes
- Chapter markers (see chapters.json)

### Cuts to Make
- Long pauses while thinking
- Errors/mistakes (unless showing for teaching)
- Excessive "um" or "uh"
