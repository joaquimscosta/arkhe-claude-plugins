# Stitch Prompt Template

Use this structure when generating optimized Google Stitch prompts.

---

**Prompt Objective:**  
[Short sentence describing the purpose — e.g., “Design a mobile e-commerce app homepage.”]

---

**Instructions:**  
Describe *what to create*, using clear UI nouns and structure cues.

Example pattern:
- Include: [Header, Sidebar, Main content, Footer]
- Add: [Cards, Table, Filters, CTA Buttons]
- Position elements: [above/below/left/right as needed]
- Emphasize responsiveness and component reuse.

---

**Visual Style:**  
List 3–6 style cues defining the look and feel.

Example:
- Modern, minimal, professional
- Deep blue accents, light neutral background
- Rounded sans-serif typography
- Subtle shadows and spacing based on 8px grid

---

**Constraints:**  
- Screen: [web / mobile / tablet]
- Layout: [single-column / 2-column / responsive]
- Export: [Figma + front-end code]

---

**Example Final Output:**

> Design a responsive web dashboard for a project management tool.  
> Include: left sidebar (logo, navigation links), top header (search, notifications, user avatar),  
> main section with project cards and progress chart.  
> Style: clean, professional, muted blue palette, modern sans-serif, light neutral background.  
> Optimize for desktop; maintain consistent 8px spacing and component reuse for export.

---

**Note:**  
Always output one atomic, Stitch-ready prompt per request.
Avoid meta commentary, reasoning text, or multiple prompts in a single response.
